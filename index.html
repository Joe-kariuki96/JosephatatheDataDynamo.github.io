"Don'tGetKicked"- Automotive-Insights-in-Auto-Auction-Predictions

The "Don'tGetKicked" dataset, sourced from Kaggle, includes data on auto auctions, comprising 72,983 rows with 34 columns for training and 48,707 rows with 33 columns for testing. Machine learning techniques was applied to predict problematic vehicles ("kicked" cars) in auctions, assisting dealerships in decision-making. Data preprocessing included imputation, scaling, and encoding, and models such as logistic regression, random forest, and XGBoost were trained. The random forest model achieved the highest accuracy (90.32%), identifying key features like wheel type and vehicle age as influential. This study underscores machine learning's potential in enhancing automotive auction processes and financial risk management for dealerships.

Classifying-images-from-Fashion-MNIST-using-feedforward-neural-networks

This project implemented image classification on the Fashion MNIST dataset, containing 60,000 training images and 10,000 testing images across ten clothing and accessory classes. The dataset was preprocessed, split, and loaded using data loaders, with optional data augmentation for model generalization. The neural network included an input layer, two hidden layers with 16 and 32 neurons, and an output layer with ten classes. Training employed stochastic gradient descent with accuracy and loss monitored for validation. After ten epochs, the model reached a validation accuracy of 85.29%, showcasing effective classification. The project demonstrated the potential of deep learning for fashion image recognition.

Text-Classification-Model-Evaluation-with-Bag-of-Words

This report evaluates a text classification model's performance, emphasizing the F1 score as a key metric over traditional accuracy. The analysis focuses on assessing the model's effectiveness, particularly in cases of class imbalance. Data preparation involved tokenization, lowercasing, stop-word removal, and stemming. The model achieved an F1 score of 0.4458, indicating moderate predictive capability. In contrast, the F1 scores for zero predictions and random guesses were 0.0 and 0.1090, respectively. The findings highlight the F1 score's value in providing a nuanced understanding of model performance, suggesting the need for further improvements.
